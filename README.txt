For this project, you'll be writing a program that plays sudoku.  The files you read will have the following type of format:
1 - -  
- - 2
3 - -
Where a - indicates a blank line and a number indicates a number.  The input boards should be 9x9 size.
Your program should run with the following command:
python CSP.py input_file.txt
And should output the completed puzzle to standard in with the same format.
